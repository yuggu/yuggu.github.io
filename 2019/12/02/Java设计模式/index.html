单例模式

  单例模式的存在为了保证一个类仅有一个实例；

  比如对于某一个场景，我们经常使用某个对象的某个方法，那么对于每一个用户都需要创建一个对象（比如对于百度的搜索引擎，百度是对象，可能存在很多用户同时使用它的搜索引擎，那么就需要同时创建很多对象，但其实我们只需要使用这个搜索的方法），这其实会造成很大的浪费；

  那么我们能不能只创建一个对象，使得所有用户共用它呢；这就是单例模式；
